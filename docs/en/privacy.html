Privacy Policy

Last updated: 2026-06-21

QiaoBox is a local macOS app. File processing happens on your Mac by default, and the app accesses files only when you choose to open, preview, convert, or export them.

In-App Data

  • QiaoBox does not upload your documents, images, QR code content, or calculation input.
  • QiaoBox does not collect MAC addresses, disk serial numbers, or other hardware identifiers.
  • File access is limited to actions you start, such as opening, previewing, converting, and exporting files.

Website Analytics

This website uses Cloudflare Web Analytics to understand page visits and site performance. This site does not set its own analytics cookies.

Downloads and Feedback

The installer is hosted on GitHub Releases. Download and feedback links open GitHub, where GitHub's privacy policy applies.

Contact

Please report issues through GitHub Issues.
