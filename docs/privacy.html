隐私政策

更新日期：2026-06-21

巧匣 QiaoBox 是 macOS 本地应用。应用内的文件处理默认在你的 Mac 上完成，只有在你主动选择文件、打开文档或导出内容时才访问对应文件。

应用内数据

  • 巧匣不会上传你的文档内容、图片内容、二维码内容或计算输入。
  • 巧匣不会采集 MAC 地址、硬盘序列号或其它硬件唯一标识。
  • 文件访问仅用于你主动选择的打开、预览、转换和导出操作。

网站统计

本网站使用 Cloudflare Web Analytics 统计页面访问情况，用于了解发布页的访问量和页面表现。该统计不使用本站自设 Cookie。

首页可能显示由 visitor-badge.laobi.icu 提供的第三方访问量徽章。该徽章通过加载远程 SVG 图片累加页面访问次数，不保存你在应用内处理的文件内容或输入内容。

下载与反馈

安装包通过 GitHub Releases 托管。下载和反馈链接会跳转到 GitHub，相关访问会受 GitHub 隐私政策约束。
